# 历史文档归档

本目录只保存已被替代的设计与实施记录，便于追溯决策背景。归档文档不是当前需求、运行时
契约或待执行计划；其中的命令、复选框、migration、legacy compatibility 和 TV execution
描述均不得直接实施。

当前实现以[架构收敛 PRD](../superpowers/specs/2026-07-11-project-architecture-convergence-prd.md)、
[ADR 0001](../adr/0001-standalone-detail-routes.md)、
[ADR 0002](../adr/0002-subscription-state-convergence.md)和
[ADR 0003](../adr/0003-latest-subscription-storage-only.md)为准。

归档按原文档类型分为：

- [`superpowers/specs/`](superpowers/specs/)：已被替代的需求与设计。
- [`superpowers/plans/`](superpowers/plans/)：已被替代、不可执行的实施计划。
